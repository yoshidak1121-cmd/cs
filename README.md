# 顧客満足度入力Webアプリ (NC-CSAT)

関係会社から顧客満足度情報を収集するための静的Webアプリです。  
GitHub Pages で動作し、サーバ不要でメール本文にJSONを埋め込んで送信します。

## フォルダ構成

```
/
├── index.html   # メイン画面
├── style.css    # スタイルシート
├── app.js       # アプリロジック
├── sample.json  # 出力JSONサンプル
└── README.md    # このファイル
```

## 機能

| 機能 | 説明 |
|------|------|
| Survey ID 自動生成 | ページ読み込み時に `YYYYMMDD-HHMMSS` 形式で生成 |
| 固定値 | SurveyYear=2025, FiscalHalf=2H |
| 総合評価入力 | 7項目を1〜5点で評価 |
| 根拠案件登録 | 5点以外の評価には根拠案件の紐づけが必須 |
| バリデーション | 入力漏れ・整合性チェック＋エラー強調表示 |
| JSON生成 & 送信 | `mailto:` でメーラーを起動、JSON本文を挿入 |
| Revision管理 | 修正再提出時にリビジョン番号を加算 |

## 使い方

1. `index.html` をブラウザで開く（またはGitHub Pagesで公開する）
2. サービスセンター名を入力する
3. 総合評価（7項目）をスコアで選択する
4. 5点以外の項目については「案件を追加」ボタンで根拠案件を登録し、評価項目に紐づける
5. 「入力チェック」でバリデーション確認
6. 「送信（メール作成）」でメーラーが起動する。内容を確認して送信する

## 出力JSONフォーマット

`sample.json` を参照してください。

## 送信先メールアドレス

`app.js` 内の `FIXED.MAIL_TO` を変更してください（デフォルト: `nc-csat@xxxxx.co.jp`）。
